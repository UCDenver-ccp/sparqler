query

macro

(query q & body)

Let you craft a SPARQL-query by composing functions

ex: (query (select :s) (where :s :p :p))

Which will return a valid SPARQL 1.1 string: ‘SELECT ?s WHERE { ?s ?p ?o }’

The macro can also be called with a query-map as its first argument, allowing you to work on saved queries.
