encode

(encode x)

Encodes keywords to ?-prefixed variables and other values to RDF literals when applicable.

Vectors are interpreted differently according to their contents:

[keyword keyword] => prefixed name [string string] => literal^^string (typed literal) [string keyword] => string with language tag [keyword] => - to be used with BASE [map keyword] => (:content map) AS keyword :keyword => _:keyword (blank node) [[]] => []

Maps are expaned and compiled according to its contents, tag, bounds and separator
